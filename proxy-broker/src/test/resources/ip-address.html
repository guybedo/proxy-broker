Anonymous Proxy Type

An anonymous proxy type notifies that a proxy is being used, however it does not make known the real IP address of the person using this type of proxy. Because anonymous proxies do this, they can not guarantee full anonymity because it does show that a proxy is used to a third-party.

Highly Anonymous Proxy Type

A highly anonymous proxy type is also known as an elite proxy. Elite proxies are called such because they really are the best kind of proxy that you can use to protect your anonymity. Highly anonymous proxies do not show that a proxy is used nor does it reveal the real IP address of the user to a third-party.

Transparent Proxy Type

A transparent proxy type notifies that you are using a proxy and passes the real IP address of the user in HTTP headers to a third-party. Transparent proxy use is very risky since it does not provide an anonymous connection.
